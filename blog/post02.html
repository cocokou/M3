JavaScript按值传递

传递的是实际参数的一个拷贝，而不是参数本身；

var age = 18;
function addDecade(x) {
  x = x + 10;
}
addDecade(age); 
console.log(age);   // 这里age并没有改变，永远18岁，没毛病

函数内参数的变化只存在函数内，不会影响传递值给它的那位。

return

执行到return语句时，会退出函数，return之后的语句不再执行

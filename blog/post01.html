Javascript 数据类型: 字符串、数字、布尔、数组、对象、Null、Undefined

primitives: numbers, strings, booleans

objects: document, math, element, car, dog, ....

undefined

没有 return 语句，函数返回undefined；

变量没被赋值，value为undefined；

稀疏数组不存在的项, value为undefined；

不存在 / 已删除的property, value为undefined；

创建对象时, 不能将property设为undefined；

NaN

“Not a Number”, 非数字值；

NaN != NaN 自己不等于自己；

使用 isNaN() 来判断一个值是否是数字；

==

类型相同时，比较值，相同则true；

类型不同时，转换成同类型再比较；

  • 字符串和数字，将字符串转换成数字后比较；
  • 布尔值和其它，true 转换成 1 ，false 转成 0，再比较；
  • undefined == null 为true；
  • 0 == "" 为true，空字符串会转成数字 0

没有 <== 和 >== , 只有 <= 和 >= ；

就如 === 比 ==严格, !== 比 != 严格；

"a" < "b"  //按字母表顺序

"12" < 123   //字符转数字，再比较
